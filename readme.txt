Kickstart Plugin for the Katapult Theme Framework. Includes a cute ready-to-run theme.

== Description ==

The Katapult Cleanlogic plugin is a project kickstarter that ships with a cute default theme and contains all possible features you can enable in Katapult. Such as

* Templates
* Color Setup
* Sliders
* Custom Post Types
* Taxonomies
* Shortcodes

See a (demo of the Katapult Cleanlogic project here)[http://crosstec.de/.sub-wptemplatedemos/]

Katapult is a new free WordPress theme framework, or let's better say, a WordPress theme kickstarter. It contains all the usual stuff you need to create your own themes more quickly and reduces all you need for a theme to just a few files.

(Katapult project and download page)[http://crosstec.de/en/wordpress/wordpress-theme-framework.html]


== Installation ==

1. Get/install/activate the Katapult theme (free)
2. Activate the Cleanlogic plugin
3. In Katapult -> Header -> CSS Select -> Choose the Cleanlogic plugin -> save
4. Check your frontend to see if the theme works
5. Adjust the plugins settings to control the Katapult Framework
6. For detailed instructions please refer to [Crosstec](http://crosstec.de/).
